Noel Tam's Data Science Journey

I started my Data Science journey on March 2018, in a 12 weeks Full Time extensive course at General Assembly. We started with basic python (Project 1), then EDA after intro of Pandas (Project 2), before we start to do full fetch data science modelling, mostly using Scikit-learn library (Project 3,4,5). Of course, that includes data processing, feature selection, cross validation and getting matrices. We did prediction using varies regression techniques like Simple linear, Multiple Linear and some regularization techniques such as lasso, Ridge and Elastic net. All this was used in Project 3. Also on the same project 3, in order to answer to a classification question, I have used Logistic, KNN, Support Vector Machine (SVM), Kernal SVM, Decision Trees and I think there are more. In Project 4, I thought I will be using Clustering technique such as K-Means, Principal Component Analysis PCA, which we have covered in GA, but sadly I did not get to use them. Instead, Project 4 focus on Web scrape, XGBoost, GridSearch, Bagging as well as NLP Natural Language Processing. Then during the later stage, while we are all preparing for capstone project, GA starts to cover deep learning and time series, but only on a intro level. However, most of us have chosen this topics for our capstone. Myself have chosen Image classification as my capstone project which uses the technique called Convolution Neural network. I have self taught keras to build CNN layers. And just to spice up the experience, I have also included Object Detection via Tensorflow's API and OpenCV to stream on webcam into my capstone project. I guess it is not possible to master all models in 12 weeks, but I think GA and the instructors have done a good job preparing us the foundation to learn new and more powerful models on our own. This we have all demostrated in our capstone projects. My journey on Data Science just started, still have a long way to go. And I hoped to fill this website with as much projects as I can. So do stay tune and walk this difficult journey with me.
